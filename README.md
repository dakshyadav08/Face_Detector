Face Detector is a computer vision project that identifies human faces in videos using the YOLOv8 model and OpenCV.
The script processes each video frame, detects faces, draws bounding boxes, and exports a new video with all detections highlighted.

This project demonstrates practical implementation of object detection and real-time video processing.
